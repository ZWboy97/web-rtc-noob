# web-rtc-noob
> 此仓库为入门学习webrtc技术之路的各种demo实现，覆盖web-rtc技术的多个常用方法和场景。
## 实现的功能或者涉及的工作
- 使用nodejs搭建http，https服务器
- 使用socket.io搭建聊天服务器，也是未来web-rtc直播中的信令服务器
- web-rtc的本地设备管理
- web-rtc采集、录制本地的音视频
- web-rtc实现本地的音视频通信
- coturn服务器的搭建
- web-rtc实现一对一P2P直播
